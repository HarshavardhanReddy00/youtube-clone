##  YouTube Clone project based on MERN development

1.This is a youtube clone project built on MongoDB,Express.js,React.js and Node.js.

## About Project

1.This allows user to see the videos.
2.Also allows to see videos based on their category.
3.Can upload videos.
4.Search box to search for their particular videos.
5.Can edit their comments if they dont like that comment.
6.User validation like username should be more than 3 characters , email must have format of @gmail.com and password must contain a capital,small,a number and special letters.
7.In backend provided JWT based validation when we login.
8.Using that we can do any thing in the backend.
9.I have used seed data to  store the dummy vedios data.
10.Using that data , i have presented in frontend.


## To run the project use

# Backend
1.First clone the project
2.run in backend as first we need to change directory as cd youtube-clone, cd backend
3.Then run npm run seed
4.npm start

# Frontend
1.change directory as cd youtube-clone,cd frontend
2.run npm run dev
3.click on link to see the UI.
